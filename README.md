## 🚀 Featured Projects

### 🦾 Vision-Guided Robotic Arm Control
Real-time control system for an **xArm Lite 6** robotic arm using computer vision. The arm detects geometric shapes (triangle, square, circle) via a **Logitech C920** webcam in eye-in-hand configuration and points to each one sequentially. Built with Python, PyQt6, OpenCV, and the xArm SDK.
